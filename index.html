A Python tool made for Roblox mod developers

This tool is originally made for mod developers, but it can be used by anyone. Here are some of it's key features:

Mod updater

Update the ImageSets for compatible mods. Click here to learn more about what makes a mod compatible.

Mod generator (Temporarily disabled)

Generate ImageSets for your mod. Temporarily disabled until I improve it.

Installation Guide

1. Download and extract the latest release

Go to the latest release to download the installer. Run the file and go through the setup process.

2. Store the program

Move the executable to where you want to store this program. It is recommended to store it in a separate folder, because it will create some files next to the executable when you run it.

3. Run the program

Run the executable.
